## ISSUE
Add document expiration gate before assignment. Assignment proceeds even if driver's CDL, medical cert, hazmat cert, or vehicle annual inspection documents are expired or missing.

## AFFECTED USER TYPES
- Driver
- Catalyst
- Compliance Officer

## FILE(S) TO MODIFY
- frontend/server/routers/dispatch.ts (assignDriver procedure)
- frontend/server/routers/documentCenter.ts

## REQUIRED CHANGE

**In dispatch.ts assignDriver():**
```typescript
async function assignDriver(loadId: number, driverId: string) {
  // Check required driver documents
  const requiredDocs = ['cdl', 'medical_certificate', 'hazmat'];

  for (const docType of requiredDocs) {
    const doc = await db.query(
      `SELECT * FROM documents WHERE user_id = $1 AND document_type = $2
       ORDER BY created_at DESC LIMIT 1`,
      [driverId, docType]
    );

    if (!doc || !doc.id) {
      throw new Error(
        `Cannot assign: Missing required document: ${docType.replace('_', ' ')}`
      );
    }

    if (new Date(doc.expirationDate) < new Date()) {
      throw new Error(
        `Cannot assign: ${docType.replace('_', ' ')} expired on ${
          doc.expirationDate.toISOString().split('T')[0]
        }`
      );
    }
  }

  // Check vehicle annual inspection
  const load = await db.query(`SELECT vehicle_id FROM loads WHERE id = $1`, [loadId]);
  const vehicleInspection = await db.query(
    `SELECT * FROM documents WHERE vehicle_id = $1 AND document_type = 'annual_inspection'
     ORDER BY created_at DESC LIMIT 1`,
    [load.vehicleId]
  );

  if (!vehicleInspection || new Date(vehicleInspection.expirationDate) < new Date()) {
    throw new Error(
      `Cannot assign: Vehicle annual inspection expired or missing`
    );
  }

  // All checks passed, proceed with assignment
  await db.query(
    `UPDATE loads SET assigned_driver_id = $1, status = 'assigned', assigned_at = NOW()
     WHERE id = $2`,
    [driverId, loadId]
  );

  return { success: true, loadId };
}
```

3. Renew medical cert → assignment succeeds
