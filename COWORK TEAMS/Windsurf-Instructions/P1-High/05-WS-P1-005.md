## ISSUE
Implement Stripe Connect OAuth callback. No endpoint to exchange Stripe Connect OAuth code for Connect account ID. Catalysts/drivers cannot connect bank accounts for payouts.

## AFFECTED USER TYPES
- Catalyst
- Driver

## FILE(S) TO MODIFY
- frontend/server/routers/stripe.ts (new endpoint)
- Schema: users table (add stripeConnectId column)

## REQUIRED CHANGE

**New endpoint (frontend/server/routers/stripe.ts):**
```typescript
app.get('/api/stripe/connect/callback', async (req: Request, res: Response) => {
  const { code, state } = req.query;
  if (!code) return res.status(400).json({ error: 'No auth code' });

  try {
    const response = await stripe.oauth.token({
      grant_type: 'authorization_code',
      code: code as string,
    });

    const stripeConnectId = response.stripe_user_id;
    await db.query(
      `UPDATE users SET stripe_connect_id = $1 WHERE id = $2`,
      [stripeConnectId, req.user.id]
    );

    res.redirect('/settings/banking?success=connected');
  } catch (error) {
    res.status(500).json({ error: 'Failed to connect Stripe account' });
  }
});
```

**Database migration:**
```sql
ALTER TABLE users ADD COLUMN stripe_connect_id VARCHAR(255);
```

4. Verify payout account is now linked
