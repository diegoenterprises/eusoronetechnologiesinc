## ISSUE
Add RBAC to compliance.ts getDashboardStats. The getDashboardStats and getExpiringItems procedures have no requireAccess with targetCompanyId. This allows cross-company compliance data access.

## AFFECTED USER TYPES
- Compliance Officer
- Safety Manager

## FILE(S) TO MODIFY
- frontend/server/routers/compliance.ts (lines 75-150)

## REQUIRED CHANGE

**Before (line 75):**
```typescript
async function getDashboardStats(userId: string) {
  const stats = await db.query(`SELECT * FROM compliance_stats`);
  return stats;
}
```

**After:**
```typescript
async function getDashboardStats(userId: string, req: Request) {
  await requireAccess(req, 'compliance_officer', targetCompanyId);
  const user = await getUser(userId);
  const stats = await db.query(
    `SELECT * FROM compliance_stats WHERE company_id = $1`,
    [user.companyId]
  );
  return stats;
}
```

Apply same pattern to getExpiringItems (add WHERE company_id = $1 filter).
