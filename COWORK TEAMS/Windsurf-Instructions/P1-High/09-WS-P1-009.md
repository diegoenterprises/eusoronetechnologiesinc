## REQUIRED CHANGE

**Pattern (from ShipperLoads.tsx):**
```typescript
const [error, setError] = useState<string | null>(null);

useEffect(() => {
  const fetchData = async () => {
    try {
      setError(null);
      const data = await apiCall();
      setData(data);
    } catch (err) {
      setError(err.message || 'Failed to load data');
    }
  };
  fetchData();
}, []);

return (
  <div>
    {error && (
      <ErrorBoundary message={error}>
        <button onClick={() => window.location.reload()}>
          Retry
        </button>
      </ErrorBoundary>
    )}
    {!error && <YourContent />}
  </div>
);
```

**Apply to all pages:**
- ComplianceDashboard.tsx
- DispatchDashboard.tsx
- AllLoadsList.tsx
- BidsPage.tsx
- (and all other list/dashboard pages)
