## ISSUE
Add refund wallet reconciliation. payments.ts processRefund() calls Stripe refund API but doesn't update wallet balances. Wallet shows money that's been refunded.

## AFFECTED USER TYPES
- Shipper
- Catalyst

## FILE(S) TO MODIFY
- frontend/server/routers/payments.ts
- frontend/server/routers/wallet.ts

## REQUIRED CHANGE

**In payments.ts processRefund():**
```typescript
async function processRefund(chargeId: string, amount: number, reason: string) {
  try {
    // 1. Create refund with Stripe
    const refund = await stripe.refunds.create({
      charge: chargeId,
      amount: Math.round(amount * 100), // cents
      reason: reason as 'duplicate' | 'fraudulent' | 'requested_by_customer',
    });

    if (refund.status !== 'succeeded') {
      throw new Error('Stripe refund failed');
    }

    // 2. Get original charge/payment record
    const payment = await db.query(
      `SELECT user_id, amount FROM payments WHERE stripe_charge_id = $1`,
      [chargeId]
    );

    // 3. Debit user wallet by refund amount
    await db.query(
      `INSERT INTO wallet_transactions
       (user_id, amount, type, stripe_refund_id, reference_id)
       VALUES ($1, $2, 'refund_debit', $3, $4)`,
      [payment.userId, -amount, refund.id, chargeId]
    );

    // 4. Update wallet balance
    const wallet = await db.query(
      `SELECT available_balance FROM wallets WHERE user_id = $1`,
      [payment.userId]
    );

    const newBalance = wallet.availableBalance - amount;

    await db.query(
      `UPDATE wallets SET available_balance = $1 WHERE user_id = $2`,
      [newBalance, payment.userId]
    );

    // 5. Record refund
    await db.query(
      `UPDATE payments SET refund_id = $1, refund_amount = $2, refund_date = NOW()
       WHERE stripe_charge_id = $3`,
      [refund.id, amount, chargeId]
    );

    return refund;
  } catch (error) {
    console.error('Refund processing failed:', error);
    throw error;
  }
}
```
