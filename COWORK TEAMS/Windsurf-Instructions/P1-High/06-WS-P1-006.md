## ISSUE
Wire fee deduction into load checkout. stripe.ts createLoadCheckout doesn't use feeCalculator. Platform fees are never charged on checkout.

## AFFECTED USER TYPES
- Shipper
- Super Admin

## FILE(S) TO MODIFY
- frontend/server/routers/stripe.ts (createLoadCheckout procedure)

## REQUIRED CHANGE

**Before:**
```typescript
async function createLoadCheckout(loadId: number, amount: number) {
  const session = await stripe.checkout.sessions.create({
    payment_method_types: ['card'],
    line_items: [{ price_data: { amount }, quantity: 1 }],
    mode: 'payment',
  });
  return session;
}
```

**After:**
```typescript
async function createLoadCheckout(loadId: number, amount: number) {
  const fee = feeCalculator.calculateFee(amount);
  const total = amount + fee;

  const session = await stripe.checkout.sessions.create({
    payment_method_types: ['card'],
    line_items: [{ price_data: { amount: total * 100 }, quantity: 1 }],
    mode: 'payment',
    application_fee_amount: Math.round(fee * 100),
  });

  await db.query(
    `INSERT INTO platform_revenue (load_id, fee_amount, stripe_session_id)
     VALUES ($1, $2, $3)`,
    [loadId, fee, session.id]
  );

  return session;
}
```

## VERIFICATION
1. Shipper creates load checkout for $1000
2. Verify Stripe session shows total = $1000 + platform fee
3. Verify platform_revenue records fee entry
4. Complete payment → fee deducted to platform account
