# WS-P0-018: Insurance Enforcement at Bid Acceptance

**Priority:** P0-CRITICAL
**Team:** Delta (Compliance & Regulatory)
**Estimated Complexity:** Medium
**Unlocks:** Safety compliance, carrier vetting, hazmat authorization

## Problem
The `accept()` mutation in `loadBidding.ts` does not validate insurance before accepting a bid. The auto-accept rules support a `requiredInsuranceMin` field (line 416), but this is only a rule definition — it is NOT enforced at acceptance time. The `insurance.ts` router has comprehensive insurance management but is completely isolated from the bid acceptance flow. A carrier with zero insurance can haul hazmat crude oil.

## What to Implement

Wire insurance verification into the bid acceptance flow:

```typescript
// Create: frontend/server/services/compliance/insuranceGate.ts

export async function verifyCarrierInsurance(
  carrierId: number,
  companyId: number,
  load: { cargoType: string; estimatedValue: number; hazmat: boolean }
) {
  const certs = await db.select()
    .from(insuranceCertificates)
    .where(and(
      eq(insuranceCertificates.companyId, companyId),
      eq(insuranceCertificates.status, 'active'),
      gte(insuranceCertificates.expirationDate, new Date()),
    ));

  if (!certs.length) {
    return { passed: false, reason: 'No valid insurance certificates on file' };
  }

  // Check for required coverage types
  const hasAutoLiability = certs.some(c => c.coverageType === 'auto_liability');
  const hasCargoInsurance = certs.some(c => c.coverageType === 'cargo');

  if (!hasAutoLiability) {
    return { passed: false, reason: 'Missing auto liability insurance' };
  }
  if (!hasCargoInsurance) {
    return { passed: false, reason: 'Missing cargo insurance' };
  }

  // For hazmat loads, require hazmat endorsement
  if (load.hazmat || load.cargoType?.toLowerCase().includes('crude')) {
    const hasHazmat = certs.some(c =>
      c.coverageType === 'hazmat' ||
      c.endorsements?.includes('hazmat')
    );
    if (!hasHazmat) {
      return { passed: false, reason: 'Hazmat load requires hazmat insurance endorsement' };
    }
  }

  // Check minimum coverage amount ($1M default)
  const minCoverage = 1_000_000;
  const maxCoverage = Math.max(...certs.map(c => c.coverageAmount || 0));
  if (maxCoverage < minCoverage) {
    return {
      passed: false,
      reason: `Insurance coverage ($${maxCoverage.toLocaleString()}) below minimum ($${minCoverage.toLocaleString()})`,
    };
  }

  return { passed: true, certificates: certs };
}
```

Then call this in `loadBidding.ts` accept() mutation before assignment.
