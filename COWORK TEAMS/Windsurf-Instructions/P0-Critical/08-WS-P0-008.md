## ISSUE

`loadBidding.ts submit()` creates a bid in the database but NEVER calls `emitBidReceived()`. Shippers don't get real-time bid notifications — they must manually refresh the page to see new bids. Same issue with counter-offers. The import for `emitBidReceived` exists in `loads.ts` but is never used in the bidding router.

## AFFECTED USER TYPES

Shipper (waiting for bids), Catalyst/Carrier (submitting bids), Broker (managing bids)

## AFFECTED WORKFLOW

Carrier submits bid → **Real-time notification to shipper** → Shipper reviews bid

## FILE(S) TO MODIFY

`frontend/server/routers/loadBidding.ts` — `submit` procedure (~line 244) and `counter` procedure

## REQUIRED CHANGE

```typescript
// In loadBidding.submit(), after successful bid insert:
import { emitBidReceived, emitNotification } from '../_core/websocket';

// After: await db.insert(loadBids).values({...})
emitBidReceived({
  loadId: input.loadId,
  bidId: newBidId,
  bidAmount: input.amount,
  bidderUserId: ctx.user.id,
  bidderName: ctx.user.name || 'Anonymous',
  bidderRole: ctx.user.role,
  bidderCompanyName: company?.name,
  timestamp: new Date().toISOString(),
});

// Notify the load owner (shipper)
const [load] = await db.select().from(loads).where(eq(loads.id, input.loadId));
if (load?.shipperId) {
  emitNotification({
    userId: load.shipperId,
    type: 'new_bid',
    title: 'New Bid Received',
    body: `${ctx.user.name || 'A carrier'} bid $${input.amount.toLocaleString()} on load ${load.loadNumber}`,
    data: { loadId: input.loadId, bidId: newBidId, amount: input.amount },
  });
}

// In loadBidding.counter(), after counter-offer insert:
emitBidReceived({
  loadId: bid.loadId,
  bidId: bid.id,
  bidAmount: input.counterAmount,
  isCounterOffer: true,
  counterFromUserId: ctx.user.id,
  timestamp: new Date().toISOString(),
});
```

Also add emission in `accept()` and `reject()`:

```typescript
// In accept(): notify bidder their bid was accepted
emitNotification({
  userId: bid.bidderUserId,
  type: 'bid_accepted',
  title: 'Bid Accepted!',
  body: `Your bid of $${bid.amount} on load ${load.loadNumber} was accepted`,
  data: { loadId: bid.loadId, bidId: bid.id },
});

// Emit bid:awarded event
emitBidAwarded({
  loadId: bid.loadId,
  bidId: bid.id,
  winnerId: bid.bidderUserId,
  amount: bid.amount,
});

// In reject(): notify bidder
emitNotification({
  userId: bid.bidderUserId,
  type: 'bid_rejected',
  title: 'Bid Not Selected',
  body: `Your bid on load ${load.loadNumber} was not selected`,
  data: { loadId: bid.loadId, bidId: bid.id },
});
```
