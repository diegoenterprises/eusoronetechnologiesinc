## THE TEST

Execute a complete load lifecycle from creation to payment:

### Step 1: SHIPPER CREATES LOAD
- Login as shipper@eusotrip.com
- Create new load:
  - Cargo: Crude Oil (hazmat class 3, UN1267)
  - Pickup: Houston, TX (Eusorone terminal)
  - Delivery: Austin, TX
  - Rate: $5,000
  - Weight: 45,000 lbs / 200 bbl
- ESANG AI should assist with classification
- VERIFY: Load appears in marketplace with status "posted"
- VERIFY: `search_loads({ status: "posted" })` returns the load

### Step 2: CATALYST SUBMITS BID
- Login as catalyst@eusotrip.com
- Browse load marketplace
- Find the Houston→Austin load
- Submit bid: $4,800
- VERIFY: Shipper receives real-time WebSocket notification (WS-P0-008)
- VERIFY: `get_load_details({ loadId: X })` shows bid in bids array

### Step 3: SHIPPER ACCEPTS BID
- Login as shipper@eusotrip.com
- View bids on load
- Accept catalyst's $4,800 bid
- VERIFY: Insurance gate passes (WS-P0-006) — carrier has sufficient insurance
- VERIFY: Load status → "awarded"
- VERIFY: Catalyst receives bid:accepted notification
- VERIFY: WebSocket event emitted (WS-P0-007)

### Step 4: CATALYST ASSIGNS DRIVER + VEHICLE
- Login as catalyst@eusotrip.com
- Open awarded load
- Assign driver@eusotrip.com + Peterbilt 389 + MC-407 tanker
- VERIFY: Compliance gate passes (WS-P0-005):
  - Driver CDL valid
  - Vehicle inspection current
  - Carrier authority active
  - Insurance sufficient
- VERIFY: Load status → "assigned"
- VERIFY: Driver receives notification

### Step 5: DRIVER PICKS UP
- Login as driver@eusotrip.com (or simulate)

- Arrive at pickup location
- Complete pre-trip inspection
- Execute pickup (BOL signature)
- VERIFY: Load status → "in_transit"
- VERIFY: GPS tracking starts (positions every 30 sec)
- VERIFY: Shipper dashboard shows real-time position

### Step 6: DRIVER DELIVERS
- Driver arrives at delivery location
- Complete delivery (POD capture)
- VERIFY: Load status → "delivered"
- VERIFY: WebSocket event fires

### Step 7: SETTLEMENT + PAYMENT
- Settlement auto-creates (WS-P0-004):
  - Load rate: $4,800
  - Platform fee (5%): $240

- VERIFY: Platform revenue records $240 fee
