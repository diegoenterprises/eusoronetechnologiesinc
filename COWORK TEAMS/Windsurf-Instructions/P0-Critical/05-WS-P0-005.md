## ISSUE

Loads can be assigned to drivers/carriers WITHOUT checking:
- Insurance is active and sufficient for cargo type
- FMCSA safety status (authority active, no OOS orders)
- Driver has valid CDL with required endorsements (hazmat H, tanker N)
- Vehicle inspection is current
- TWIC card validity (for terminal access)

This is a DOT regulatory violation. 49 CFR 387.9 requires minimum $5M insurance for hazmat loads. Allowing uninsured/non-compliant carriers to haul crude oil loads is a liability disaster.

The compliance engine EXISTS (50+ document types, FMCSA data, insurance monitoring) — it just isn't ENFORCED at the assignment point.

## AFFECTED USER TYPES

Catalyst (assigns loads), Driver (assigned), Shipper (liability), Compliance Officer, Safety Manager, Admin

## AFFECTED WORKFLOW

Load Assignment: Catalyst assigns driver + vehicle → compliance gate MUST pass → assignment confirmed

### In dispatch.ts assignLoad():

Before the actual assignment, add compliance gate:

```typescript
// === COMPLIANCE GATE — BLOCKS NON-COMPLIANT ASSIGNMENTS ===

// 1. Get load details (need hazmat class, cargo type)
const [load] = await db.select().from(loads).where(eq(loads.id, input.loadId));
if (!load) throw new TRPCError({ code: 'NOT_FOUND', message: 'Load not found' });

// 2. Get driver details
const [driver] = await db.select().from(users).where(eq(users.id, input.driverId));
if (!driver) throw new TRPCError({ code: 'NOT_FOUND', message: 'Driver not found' });

// 3. Get carrier company
const companyId = driver.companyId;
if (!companyId) throw new TRPCError({ code: 'PRECONDITION_FAILED', message: 'Driver must belong to a company' });

const [company] = await db.select().from(companies).where(eq(companies.id, companyId));

// 4. CHECK: FMCSA Authority Status
if (company?.dotNumber) {
  const safetyData = await getCarrierSafetyIntel(company.dotNumber);
  if (safetyData?.authorityStatus === 'INACTIVE' || safetyData?.authorityStatus === 'REVOKED') {
    throw new TRPCError({
      code: 'FORBIDDEN',
      message: `Carrier authority is ${safetyData.authorityStatus}. Cannot assign load.`,
    });
  }
  // Check for active Out-of-Service orders
  const oosStatus = await getOOSStatus(company.dotNumber);
  if (oosStatus?.hasActiveOOS) {
    throw new TRPCError({
      code: 'FORBIDDEN',
      message: 'Carrier has active Out-of-Service order. Cannot assign load.',
    });
  }
}

// 5. CHECK: Insurance minimums for hazmat
const HAZMAT_INSURANCE_MINIMUMS: Record<string, number> = {
  '1': 5000000,   // Explosives
  '2': 5000000,   // Gases
  '3': 5000000,   // Flammable liquids (crude oil, gasoline)
  '4': 5000000,   // Flammable solids
  '5': 5000000,   // Oxidizers
  '6': 5000000,   // Poisons
  '7': 5000000,   // Radioactive
  '8': 5000000,   // Corrosives
  '9': 1000000,   // Misc
};

if (load.hazmatClass) {
  const requiredInsurance = HAZMAT_INSURANCE_MINIMUMS[load.hazmatClass] || 1000000;

  // Query carrier's insurance from FMCSA data or platform documents
  const insuranceAmount = await getCarrierInsuranceAmount(companyId, company?.dotNumber);

  if (!insuranceAmount || insuranceAmount < requiredInsurance) {
    throw new TRPCError({
      code: 'FORBIDDEN',
      message: `Insufficient insurance. Hazmat Class ${load.hazmatClass} requires $${(requiredInsurance/1000000).toFixed(0)}M coverage. Carrier has $${insuranceAmount ? (insuranceAmount/1000000).toFixed(1) : '0'}M.`,
    });
  }
}

// 6. CHECK: Driver endorsements
// For hazmat loads: driver needs H endorsement
// For tanker loads: driver needs N endorsement
if (load.hazmatClass) {
  const driverDocs = await db.select().from(documents)
    .where(and(eq(documents.userId, input.driverId), eq(documents.type, 'cdl')));

  // Check if CDL has hazmat (H) and tanker (N) endorsements
  // and that they're not expired
  const cdl = driverDocs[0];
  if (!cdl) {
    throw new TRPCError({
      code: 'FORBIDDEN',
      message: 'Driver has no CDL on file. Upload CDL before hazmat assignment.',
    });
  }
  if (cdl.expiresAt && new Date(cdl.expiresAt) < new Date()) {
    throw new TRPCError({
      code: 'FORBIDDEN',
      message: 'Driver CDL is expired. Cannot assign hazmat load.',
    });
  }
}

// 7. CHECK: Vehicle inspection current
if (input.vehicleId) {
  const [vehicle] = await db.select().from(vehicles).where(eq(vehicles.id, input.vehicleId));
  if (vehicle?.lastInspectionDate) {
    const daysSinceInspection = Math.floor((Date.now() - new Date(vehicle.lastInspectionDate).getTime()) / 86400000);
    if (daysSinceInspection > 365) {
      throw new TRPCError({
        code: 'FORBIDDEN',
        message: `Vehicle annual inspection expired (${daysSinceInspection} days since last inspection). Max 365 days.`,
      });
    }
  }
}

// === ALL GATES PASSED — PROCEED WITH ASSIGNMENT ===
// Log compliance check pass in audit trail
await db.insert(auditLogs).values({
  action: 'COMPLIANCE_GATE_PASSED',
  userId: ctx.user.id,
  details: JSON.stringify({
    loadId: input.loadId,
    driverId: input.driverId,
    vehicleId: input.vehicleId,
    hazmatClass: load.hazmatClass,
    checksPerformed: ['authority', 'oos', 'insurance', 'endorsements', 'inspection'],
  }),
});
```
