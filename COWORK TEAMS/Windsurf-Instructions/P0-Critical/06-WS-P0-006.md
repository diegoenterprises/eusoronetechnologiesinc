## ISSUE

`loadBidding.ts accept()` proceeds without confirming the bidding carrier has required insurance coverage. For crude oil and hazmat loads, 49 CFR 387.9 requires $5M liability coverage. Accepting an underinsured bid creates immediate regulatory violation when the load ships.

## AFFECTED USER TYPES

Shipper (accepts bid), Catalyst/Carrier (submits bid), Compliance Officer, Safety Manager

## AFFECTED WORKFLOW

Shipper receives bid → Reviews bid → Accepts bid → **Insurance verification MUST happen here**

## FILE(S) TO MODIFY

`frontend/server/routers/loadBidding.ts` — the `accept` procedure

## REQUIRED CHANGE

In the `accept` mutation, before setting bid status to 'accepted':

```typescript
// Before accepting the bid, verify carrier insurance
const [load] = await db.select().from(loads).where(eq(loads.id, bid.loadId));
const [bidder] = await db.select().from(users).where(eq(users.id, bid.bidderUserId));

if (bidder?.companyId) {
  const [company] = await db.select().from(companies).where(eq(companies.id, bidder.companyId));

  if (load?.hazmatClass && company?.dotNumber) {
    const requiredMin = 5000000; // $5M for hazmat (49 CFR 387.9)
    const insuranceAmount = await getCarrierInsuranceAmount(bidder.companyId, company.dotNumber);

    if (!insuranceAmount || insuranceAmount < requiredMin) {
      throw new TRPCError({
        code: 'FORBIDDEN',
        message: `Cannot accept bid: carrier insurance ($${((insuranceAmount || 0) / 1000000).toFixed(1)}M) below required minimum ($${(requiredMin / 1000000).toFixed(0)}M) for Hazmat Class ${load.hazmatClass} cargo.`,
      });
    }
  }
}

// Proceed with bid acceptance...
```
