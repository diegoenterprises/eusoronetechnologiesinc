## ISSUE

No settlement automation exists. When a load reaches "delivered" status, nothing triggers payment to the catalyst/carrier. Driver earnings accumulate in display but are never formally settled. There is no cron job or event handler that closes the financial cycle.

Also: verify whether a `settlements` table exists. Our SQL probe failed — the table may be named differently or may not exist at all.

## AFFECTED USER TYPES

Catalyst/Carrier (receive settlement), Shipper (billed), Driver (earnings), Factoring (collections), Super Admin (revenue)

## AFFECTED WORKFLOW

Load Delivered → Settlement Created → Platform Fee Deducted → Carrier Wallet Credited → Load Status → PAID

## FILE(S) TO MODIFY

1. Check if settlements table exists: `SELECT table_name FROM information_schema.tables WHERE table_name LIKE '%settle%'`
2. If missing: create migration for `settlements` table
3. Create new cron job: `frontend/server/cron/settlements.ts` (or add to existing cron runner)
4. Modify: `frontend/server/routers/loadLifecycle.ts` — add settlement trigger in DELIVERED state effect

## REQUIRED CHANGE

### Step 1: Ensure settlements table exists

```sql
CREATE TABLE IF NOT EXISTS settlements (
  id INT AUTO_INCREMENT PRIMARY KEY,
  loadId INT NOT NULL,
  shipperId INT NOT NULL,
  carrierId INT NOT NULL,       -- catalyst/carrier
  driverId INT,
  loadRate DECIMAL(12,2) NOT NULL,
  platformFeePercent DECIMAL(5,2) DEFAULT 5.00,
  platformFeeAmount DECIMAL(12,2) NOT NULL,
  carrierPayment DECIMAL(12,2) NOT NULL,
  accessorialCharges DECIMAL(12,2) DEFAULT 0.00,
  totalShipperCharge DECIMAL(12,2) NOT NULL,
  status ENUM('pending','processing','completed','failed','disputed') DEFAULT 'pending',
  stripeTransferId VARCHAR(255),
  settledAt TIMESTAMP,
  createdAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updatedAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  FOREIGN KEY (loadId) REFERENCES loads(id),
  UNIQUE INDEX idx_settlement_load (loadId)
);
```

### Step 2: Add settlement trigger in load lifecycle

In `loadLifecycle.ts`, when a load transitions to DELIVERED state:

```typescript
// In the DELIVERED state effect handler:
async function onLoadDelivered(loadId: number, db: Database) {
  const [load] = await db.select().from(loads).where(eq(loads.id, loadId));
  if (!load) return;

  // Calculate settlement
  const fee = await feeCalculator.calculateFee({
    transactionType: 'load_booking',
    amount: Number(load.rate),
  });

  const platformFee = fee.finalFee;
  const carrierPayment = Number(load.rate) - platformFee;

  // Create settlement record
  await db.insert(settlements).values({
    loadId: load.id,
    shipperId: load.shipperId,
    carrierId: load.catalystId || load.driverId,
    driverId: load.driverId,
    loadRate: load.rate,
    platformFeePercent: fee.feePercent || 5.00,
    platformFeeAmount: platformFee,
    carrierPayment: carrierPayment,
    totalShipperCharge: Number(load.rate),
    status: 'pending',
  });

  // Credit carrier wallet
  const carrierId = load.catalystId || load.driverId;
  if (carrierId) {
    const wallet = await ensureWallet(db, carrierId);
    await db.update(wallets)
      .set({ availableBalance: sql`${wallets.availableBalance} + ${carrierPayment}` })
      .where(eq(wallets.userId, carrierId));

    // Record transaction
    await db.insert(walletTransactions).values({
      walletId: wallet.id,
      userId: carrierId,
      type: 'earnings',
      amount: carrierPayment,
      description: `Settlement for load ${load.loadNumber}`,
      referenceType: 'load',
      referenceId: String(load.id),
    });
  }

  // Record platform revenue
  await db.insert(platformRevenue).values({
    transactionType: 'load_settlement',
    grossAmount: load.rate,
    platformShare: platformFee,
    netToUser: carrierPayment,
    loadId: load.id,
  });
}
```

### Step 3: Optional — cron job for batch settlements

For loads that slipped through without auto-settlement:

```typescript
// Run every hour: settle any DELIVERED loads without settlements
const unsettledLoads = await db.select()
  .from(loads)
  .where(and(
    eq(loads.status, 'delivered'),
    notExists(db.select().from(settlements).where(eq(settlements.loadId, loads.id)))
  ));

for (const load of unsettledLoads) {
  await onLoadDelivered(load.id, db);
}
```
